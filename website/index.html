Frankenstein is a record and play Functional Testing tool for Swing applications. Frankenstein's focus is on readable, simple functional tests that can be shared by everyone on a team.

You could use Frankenstein for the following:

  • Automate acceptance tests
  • Record bugs so that they may be easily reproduced later
  • Automated demos

Features:

  • Test scripts are independent of widget location
  • Supports most major swing components
  • Easy customization / extension
  • Easy integration
  • Support for testing multithreaded applications
  • HTML reports
  • Ruby driver
  • OGNL based assertions
  • Performance: Designed to minimize the need for explicit delays in tests
  • Supports JDK 1.4+
