Module FrankensteinDriver
In: frankenstein_driver.rb

The Frankenstein driver allows a Swing user interface to be tested. Regular expressions can be specified using the regular Ruby syntax.

Public Instance methods

activate_dialog(title)

Activates a dialog with a specified title. The title can be specified as a regular expression

activate_internal_frame(title)

Activates an internal frame. Regular expressions are supported

activate_window(title)

Activates a window with a specified title. The title can be specified as a regular expression

assert(component_name,ognl_expression,expected_value)

Checks whether an OGNL expression evaluated against a specified component matches the specified value. The expected value can be specified as a regular expression.

assert_checkbox_not_selected(checkbox_name)

Check whether a check box with the specified name is not selected

assert_checkbox_selected(checkbox_name)

Check whether a check box with the specified name is selected

assert_disabled(component_name)

Check whether a component with the specified name is enabled

assert_enabled(component_name)

Check whether a component with the specified name is enabled

assert_false(component_name,ognl_expression)

Checks whether the specified OGNL expression is false.

assert_number_of_table_rows(table_name, number_of_rows)

Check the number of rows displayed in a table

assert_radiobutton_not_selected(radiobutton_name)

Check whether a radio button with the specified name is not selected

assert_radiobutton_selected(radiobutton_name)

Check whether a radio button with the specified name is selected

assert_table_cell(table_name, row, column, value)

Check the value of a specified table cell. The cell value can be specified as a regular expression

assert_table_row(table, row, *cell_values)

Check the elements of a specified table row The cell values can be an arbitrary list of arguments, which be checked from left to right, starting at the first column. The cell values can be specified as regular expressions.

assert_text(text_component, expected_text)

Checks whether the text of a specified text component matches the specified value. The expected text can be specified as a regular expression.

assert_togglebutton_not_selected(togglebutton_name)

Check whether a toggle button with the specified name is not selected

assert_togglebutton_selected(togglebutton_name)

Check whether a toggle button with the specified name is selected

assert_true(component_name,ognl_expression)

Checks whether the specified OGNL expression is true.

cancel_table_edit(table)

Cancels a table edit.

click_button(button)

Click on a specified button

click_checkbox(button, value)

Click on a specified checkbox. The specified value can be true or false, and will select or deselect the checkbox

click_radiobutton(button)

Click on a specified radio button.

click_table_header(header_name,column_name)

Click on a specified column of a table header

close_all_dialogs()

Close all open dialogs recursively until no more dialogs are open. This function would normally be used at the end of a test to ensure that the next test can proceed.

close_internal_frame(title)

Close an internal frame with the specified title. The title can be specified as a regular expression.

delay(duration_in_milliseconds)

Wait for a specified duration (in milliseconds)

dialog_closed(title)

Wait for a dialog with a specified title to be closed. This function waits for 10 seconds in case the dialog is found to be open. (The wait will be made configurable in a future release). The title can be specified as a regular expression.

dialog_shown(title)

Wait for a dialog with a specified title to be opened. This function waits for 10 seconds in case the dialog is not found to be open. (The wait will be made configurable in a future release). The title can be specified as a regular expression.

double_click_list(list,item_index)

Double click on a item at the specified index of a list.

edit_table_cell(table, coords)

Edit a table cell at the specified coordinates. The coordinates are specified as a "<row>,<column>" - for example, "1,1"

enter_text(textfield, text)

Enter the specified text into a specified text field.

internal_frame_shown(title)

Checks that an internal frame with the specified title has been shown The title can be specified as a regular expression.

key_stroke(modifiers, keycode)

Enter a keystroke with the specified modifiers. This method isn‘t too tester friendly at the moment (the modifiers and the keycode are both integers). It is recommended that the recorder be used to create keystroke steps. Future releases will make it easier to specify keystrokes.

move_slider(slider,position)

Moves a specified slider to the specified position

navigate(path)

Navigate to a specified path in a menu item. Both menu bars and popup menus are supported. The path needs to be specified as a string delimited by the > character. For example: "first level>second level>item" This function does not have regular expression support.

right_click_tree(tree, path)

Right click on a tree item of the specified tree based on the specified path. The path needs to be specified as a string delimited by the > character. For example: "first level>second level>item" The path can be specified using a regular expression.

run_frankenstein_test(host,port)

Sends a test script to the Frankenstein Java runtime at the specified host and port. Waits for the test to complete, and reports test results.

select_drop_down(combo, value)

Select a specified value from a specified combo box.

select_file(path)

Select a specified file in a file chooser.

select_files(paths)

Select multiple files in a file chooser.

select_list(list, value)

Select a specified value in a specified list.

select_table_row(table, rows)

Selects the specified rows in a table. rows is a comma separated list of table rows. Example: select_table_row "table_name" , "1,2,3"

select_tree(tree, path)

Select a specified tree path. The tree path is delimited by the ’>’ character. For example: select_tree "tree_name" , "top level>second level>third level"

select_tree_with_regex(tree, *path_elements)

Select a specified tree path. Supports regular expressions For example: select_tree_with_regex "tree_name", "top level", /.*level/, "third level"

stop_table_edit(table)

Stop editing a specified table.

switch_tab(tab, title)

Switch to a tab with the specified title of a specified tabbed pane. The title can be specified as a regular expression.
